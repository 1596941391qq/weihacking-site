个人故事

我做的是在 AI 和搜索的交叉地带凿路。

六年海外 SEO，200+ 品牌从零到稳定收入。运营 hacking 周刊，拆解 SEO 暗面手法、AI 搜索策略和出海增长。公司是 Soulcraft Limited，和炬象未来（Concrete Future AI）深度合作。

从自习地图到 Pokerogue（B 站 7K+ 粉丝），从冲浪升级（三小时 Chrome 插件）到 NicheDigger（PSEO Agent，Google 高排名验证）。每一个项目都是一次 Vibe Coding 实验：用最短路径验证想法，然后决定是继续还是转向。

平台在淘汰的不是技巧，是整套假的生存环境。

Vibe Coding 实验室

每个项目都是一次快速验证实验。不是完美主义驱动的长期工程，是用最短路径验证想法，拿到真实用户反馈后再决定是否继续。三个典型速度：三个小时做 Chrome 扩展冲浪升级，一个周末搭出 NicheDigger 挖词工具原型，两周内从 PSEO Agent 概念到 SERP 首页验证。

3h
冲浪升级

Chrome 扩展，三个小时完成。游戏化记录网站使用时间，发布当天有人用。Vibe Coding 的原点：不是做完美的产品，是做能跑的东西。

1w
NicheDigger

一个周末的原型。输入品牌上下文 → 输出 100 个关键词 + 意图分类 + 竞争度。后来成为 PSEO Agent 的核心引擎，驱动了 9 个品牌的 SEO 矩阵。

2w
PSEO Agent

从概念到 SERP 首页验证只用了两周。NicheDigger 挖词 → 批量生成 → GitHub Pages 发布 → 14 天内多个关键词 SERP #1。零外链建设，纯靠内容质量。
